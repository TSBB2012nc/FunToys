# Bomb Grid — 女巫的毒药计算器

一个轻量的网页小游戏，用“标记已抽取格子”的方式实时计算 A/B 两个棋盘的胜率（条件概率）。支持移动端优先布局、点击/长按交互、胜率实时更新、游戏结束弹窗，以及中英文切换。

## 玩法说明
- 轻点格子：标记为安全（绿）。
- 长按格子：标记为炸弹（红）。
- A 与 B 是两个独立棋盘；系统会根据已标记的格子，实时计算双方胜率。

## 运行方式
直接打开 index.html 即可运行，无需构建或安装依赖。

## 主要功能
- A/B 双棋盘实时胜率计算（条件概率）
- 3x3 默认棋盘与 3 炸弹默认值
- 点击/长按交互
- 结束弹窗与动效
- 中英文切换
- 移动端优先布局

## 文件结构
- index.html — 页面结构
- styles.css — 样式与动画
- app.js — 逻辑与计算

## 说明
本项目不包含后端或数据持久化，适合作为概率游戏演示与交互实验。
